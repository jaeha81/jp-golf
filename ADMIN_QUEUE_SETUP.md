# 관리자 의뢰함 설정

코드는 준비되어 있지만, 실제 고객 의뢰를 저장하려면 Vercel 프로젝트에 두 환경변수가 필요합니다.

1. Neon(Postgres) 데이터베이스를 만들고 연결 문자열을 `DATABASE_URL`로 등록합니다.
2. 긴 임의 문자열을 만들어 `ADMIN_QUEUE_TOKEN`으로 등록합니다. 이 값은 `/admin` 화면에 입력하는 관리자 비밀번호입니다.
3. 두 변수는 Vercel의 Production과 Preview에 각각 등록한 뒤 새 배포를 만듭니다.

고객 화면의 `상담 의뢰 저장`은 `/api/admin-requests`에 저장하며, 데이터베이스가 없으면 503으로 명확히 실패합니다. 관리자 화면은 [https://jp-golf.vercel.app/admin](https://jp-golf.vercel.app/admin)에서 토큰을 입력하고 순번·고객·조건·상태를 확인할 수 있습니다. 토큰은 브라우저에 저장하지 않습니다.

상태값은 `신규 → 연락완료 → 견적안내 → 종료` 순서로 관리합니다. 개인정보 수집 동의 체크 없이는 의뢰를 제출할 수 없습니다.
