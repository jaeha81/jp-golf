## 1차 vs 2차 핵심 변경사항 (Diff)

| 항목 | 1차 (현재) | 2차 (업데이트) |
|------|-----------|--------------|
| 상담 방식 | AI가 항목별 정보 수집 | 100% 자연어 대화 상담 |
| 정보 캐치 | 폼/체크박스 기반 | 대화 안에서 AI가 자동 감지 |
| 사이트 검색 | 상담사가 락쿠텐 고라 수동 검색 | AI가 일본 사이트 직접 자동 검색 |
| 가용/가격 제시 | 상담사 → 고객 수동 안내 | AI가 자동 조회 후 직접 제시 |
| 예약 확정 | 상담사 수동 예약 | AI 자동 예약 (고객 OK 후) |
| 결제 확인 | 수동 확인 | 계좌 입금 AI 연동 → 자동 확인·발송 |
| 에이전트 범위 | 골프 부킹 전용 | 호텔·픽업·항공 에이전트 추가 예정 |
| 지역 범위 | 한국→일본 단방향 | 글로벌 교차 (한국↔일본↔중국) |

## 2차 핵심 방향 (A~E)

### A. 자연어 상담 플로우 전면 전환

```
손님 채팅 진입
  ↓
AI 자연어 대화 시작 ("사람과 대화하는 것처럼")
  ↓
대화 중 AI가 자동 감지:
  • 여행 날짜/기간
  • 라운딩 횟수·인원
  • 희망 골프장·지역
  • 선호 티오프 시간
  ↓
AI가 수집 내용을 폼으로 정리 후 손님에게 확인
  "이게 맞죠?" → 손님 OK
  ↓
AI가 일본 사이트(락쿠텐 고라 등) 자동 검색
  → 가용 티타임·가격 제시
  ↓
손님 OK → AI 자동 예약
  ↓
결제 (계좌 입금 / 카드)
  → AI ↔ 은행 연동 → 입금 확인
  → 예약 확인서 번역·자동 발송
```

### B. 단계별 에이전트 확장 구조

```
Phase 1: 골프 부킹 에이전트 (현재 구축)
Phase 2: 호텔 에이전트 추가
Phase 3: 픽업(차량) 에이전트 추가
Phase 4: 항공권 에이전트 추가
→ 모든 에이전트 동시 다발 운영
→ 패키징 → 고객이 원하는 조합 선택
```

### C. 글로벌화 로드맵

```
Level 1: 한국 → 일본 골프 (현재)
Level 2: 일본 → 한국 (역방향)
Level 3: 중국·일본·한국 3국 교차 전체 매트릭스
Level 4: 글로벌 타국 시장 확장
```

### D. 자본 전략

- 투자 유치 없이 자체 성장 → 지분 구조 단순화 유지
- 자체 금융사(여신금융) 설립 → 자본 자체 회전
- 외부 투자 불필요한 구조 목표

### E. JP Golf = JP 사업 OS

```
JP Golf 골프 부킹 (기둥 1)
  → 마스터 완료 시
  → JP 사업 전체 OS 기반 완성
  → 같은 시스템 위에 추가 사업 적재

미래 사업 스택:
1. JP Golf (현재 구축)
2. 엔터테인먼트 사업
3. 여신금융
4. AI 기반 / 데이터센터
```

## 자연어 상담 에이전트 아키텍처 설계

### 핵심 컴포넌트

```
[프론트엔드]
  • 채팅 UI (자연어 대화 인터페이스)
  • 단순·미니멀 디자인 (구글처럼 진입점 하나)

[AI 상담 에이전트 - 골프 부킹]
  • 모델: Claude / GPT (대화 품질 최우선)
  • 기능:
    - 자연어 파싱 (여행 의도 파악)
    - 정보 추출 (날짜·인원·지역·시간)
    - 확인 메시지 생성
    - 검색 쿼리 생성
    - 결과 정리·제시

[검색 레이어]
  • 락쿠텐 고라 API or 스크래핑
  • 입력: 날짜, 지역, 티오프 시간, 인원
  • 출력: 가용 골프장 목록 + 가격 + 시간

[예약 레이어]
  • 고객 OK 수신 후 자동 예약 실행
  • 예약 확인서 수신 → 번역 API → 고객 발송

[결제 레이어]
  • 은행 API 연동 (계좌 입금 감지)
  • 카드 결제 게이트웨이
  • 입금 확인 → 자동 발송 트리거

[운영 대시보드]
  • 현재 상담 현황
  • 예약 건수·매출
  • 에이전트 상태
  • 예외 처리 (AI 실패 시 상담사 알림)
```

### 구현 우선순위

```
P0: 자연어 상담 에이전트 (Claude/GPT 기반)
P0: 락쿠텐 고라 검색 연동
P1: 예약 자동화
P1: 결제 연동 (계좌 감지)
P2: 호텔/픽업/항공 에이전트 확장
P3: 글로벌화 (일본→한국 역방향)
```
