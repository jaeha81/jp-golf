# 토스페이먼츠 테스트 결제 설정

Vercel Production 환경 변수에 아래 두 값을 등록합니다. 실제 키는 저장소나 채팅에 남기지 않습니다.

| 이름 | 값 |
| --- | --- |
| `TOSS_TEST_CLIENT_KEY` | 토스페이먼츠 결제위젯 테스트 클라이언트 키 (`test_`로 시작) |
| `TOSS_TEST_SECRET_KEY` | 같은 상점의 테스트 시크릿 키 (`test_`로 시작) |

등록 후 재배포하면 상담 의뢰 저장 뒤의 `토스 테스트 결제창` 버튼이 실제 토스 테스트 결제창을 엽니다. 테스트 키로 처리된 결제는 실제 청구되지 않습니다.

결제 요청 성공 후에는 `/payment/result`에서 서버가 주문번호와 금액을 검증한 다음 토스 승인 API를 호출합니다. 클라이언트의 성공 화면만으로 결제 완료 처리하지 않습니다.
