# JP Golf 보안 진행 기록

> 상태: **기본 방어 적용 / 상용화 HOLD**  
> 기준일: 2026-07-11 (KST)

## 적용 완료

- Gemini API 키를 URL 쿼리에서 제거하고 `x-goog-api-key` 헤더로 전달
- IP 기준 best-effort rate limit: 인스턴스별 분당 20회, 키 1,000개 상한
- 메시지 최대 20개, 메시지당 2,000자, 대화 총 12,000자 제한
- Gemini 출력 최대 512 토큰 설정
- 상류 API 오류 원문을 클라이언트에 반환하지 않음

## 상용화 전 잔여 항목

- 분산 환경용 외부 rate-limit 저장소와 세션/사용자 인증
- 키 회전·운영 로그·모니터링 점검
- 실제 스테이징 API 응답 및 모바일 E2E QA
- ToS·법률 검토와 사업 책임자 승인

현재 변경은 기본 방어 수준이며, 이것만으로 공개 수익화 승인을 의미하지 않는다.
