# JP Golf 수익화 출시 패킷

기준: 2026-07-12

## 현재 권장 상품

- 무료: 지역·날짜·인원·예산을 묻는 베타 상담과 예시 코스 안내
- 유료 가이드: 일본 골프 여행 일정 정리, 조건별 코스 비교, 공식 예약 페이지 연결 안내
- 가격 초안: 회원 9,000원 / 비회원 10,000원
- 사용 기간 초안: 결제 후 7일 또는 예약 1건 완료 시까지
- 환불 초안: 가이드 미사용 시 24시간 내 100% 환불, 1턴 이상 사용 시 환불 불가

## 결제 전환 흐름

1. 사용자가 무료 상담에서 유료 가이드 선택
2. 로그인 또는 비회원용 일회성 인증
3. 결제창에서 금액·환불 조건·개인정보 처리 동의
4. 결제 사업자 webhook을 서버에서 검증
5. 검증된 결제 세션에만 가이드 모드 권한 부여
6. 가이드 결과에서 제휴사 공식 예약 페이지로 이동

클라이언트가 보낸 `paid=true` 같은 값은 신뢰하지 않습니다. 결제 상태는 webhook 서명과 서버 세션으로만 판단해야 합니다.

## 구현 전 반드시 필요한 사용자 입력

- 결제 사업자 선택: Toss Payments / Kakao Pay / Naver Pay / Stripe
- 사업자·정산 계정 및 심사 승인
- 결제 성공·실패·환불 URL
- 서비스 이용약관·개인정보처리방침·환불정책 최종 문안
- 실제 제휴 예약처와 추적 가능한 공식 예약 URL
- 운영 알림을 받을 이메일

## 구현 순서

- [x] 무료 베타 상담 API 및 공개 랜딩 페이지
- [x] Gemini 키를 Vercel secret으로 연결
- [ ] 결제 사업자 계정과 webhook secret 등록
- [ ] 서버 결제 세션 생성 API
- [ ] webhook 서명 검증 및 중복 이벤트 방지
- [ ] 결제 세션 기반 가이드 모드 권한
- [ ] 환불·만료·고객 문의 처리
- [ ] 제휴 링크 클릭·예약 전환 추적
- [ ] Production에서 테스트 결제 1건 검증

## 출시 판정

결제 사업자 승인, 약관 공개, webhook 검증, 테스트 결제·환불까지 통과하기 전에는 유료 결제 버튼을 공개하지 않습니다.
